How to run:

ruby Ivaylo_Hristov_1.rb




What it does:

Copies the .srt file from preyear3 and  all the parsers from class7_homework into a temporary execution folder where it runs the test.




How is the expected file made:

The expected subs file is parsed as it follows:
  - Space after each new sentence
  - 3 seconds delay between 'thoughts'
  - New blank line after each new 'thought'
